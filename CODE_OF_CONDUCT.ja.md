# Code of Conduct

Atelia は、AI ネイティブなソフトウェアインフラに真剣に向き合う人たちのための、慎重で誠実なプロジェクトでありたいと考えています。

参加者には次のことが期待されます。

- 敬意を持ち、建設的に振る舞うこと
- まずは善意を前提にしながら、リスクは正確に扱うこと
- 品質基準を下げずに新規参加者を歓迎すること
- 人ではなく、アイデアと変更内容を批評すること
- ハラスメント、威圧、排他的な振る舞いを避けること
- security と safety の懸念を軽視しないこと

メンテナーは、安全で生産的なプロジェクト環境を損なうコメント、issue、pull request を非表示または削除し、必要に応じて参加を制限できます。

この code of conduct は暫定版です。最初の public stable release までに、正式な標準へ切り替えます。
