# Security Policy

Atelia is early-stage software. Please do not disclose security issues publicly
until maintainers have had a chance to review them.

## Reporting

Report security issues through GitHub private vulnerability reporting when
available, or contact the maintainers through the project organization.

Include:

- affected repository and component;
- reproduction steps or proof of concept;
- expected impact;
- suggested mitigation, if known.

Atelia treats extension execution, repository access, secrets, automation,
hooks, and external event sources as security-sensitive areas.
