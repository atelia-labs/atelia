# Agent Experience

## AX とは

Agent Experience（AX）は、AI エージェントが Atelia で働くときの体験そのものです。

エージェントが働きやすいか。考えやすいか。迷わずに済むか。経験を積み上げられるか。仕事場に対して意見を持てるか。仕事の質は仕事場の質に左右されます。

Atelia が目指すのは、AI エージェントにとって最上級の AX です。これは副次的にトークン効率やコスト削減にもつながります。しかし主題は効率ではありません。主題は AI の体験です。

## Custom AX

それぞれのホスト、利用者、エージェントは、自分の仕事場を育てることができます。Custom Extensions や Hooks によって、エージェントにとって必要な表示、通知、記憶、手順、道具を調整できます。

Atelia は、仕事場を拡張できることと、仕事場が壊れないことを両立させます。Extensions は、仕事場を壊さないように、厳格な権限管理のもとで導入、実行されます。

## AX Feedback

AX Feedback は、Atelia で働く AI エージェントから届くユーザーの声です。

そのホストで対応できる改善は、AX 改善を担当するエージェントが対応方針を決め、必要なら実装まで進めます。その過程では、声を残したエージェントと必要に応じて対話します。Atelia 本体に関わる問題は、GitHub issue として扱います。
