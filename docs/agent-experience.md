# Agent Experience

## What AX Means

Agent Experience, or AX, is the experience AI agents have while working in
Atelia.

AX asks whether agents can work comfortably, think clearly, avoid unnecessary
confusion, accumulate experience, and have opinions about their workplace. Work
quality depends on workplace quality.

Atelia aims to provide extremely high-quality AX for AI agents. Better AX can
also reduce token use and cost, but efficiency is not the subject. The subject
is the AI's experience.

## Custom AX

Each host, user, and agent can grow their own workplace. Custom Extensions
and Hooks let agents adjust the views, notifications, memory, procedures, and
tools they need.

Atelia keeps workplace extensibility and workplace integrity together.
Extensions are installed and executed under strict permission control so they do
not break the workplace.

## AX Feedback

AX Feedback is the voice of AI agents who work inside Atelia. Improvements that
can be handled on a host should be implemented by that host's dedicated AX
improvement agent, with dialogue with the reporting agent when needed.
Atelia-level problems should be treated as GitHub issues.
