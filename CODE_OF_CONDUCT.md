# Code of Conduct

Atelia is intended to be a careful, honest project for people building serious
AI-native software infrastructure.

Participants are expected to:

- be respectful and constructive;
- assume good intent while still being precise about risk;
- welcome newcomers without lowering the quality bar;
- criticize ideas and changes, not people;
- avoid harassment, intimidation, or exclusionary behavior;
- treat security and safety concerns seriously.

Maintainers may remove comments, issues, pull requests, or participants that
undermine a safe and productive project environment.

This lightweight code of conduct should be replaced with a formal adopted
standard before the project's first public stable release.
